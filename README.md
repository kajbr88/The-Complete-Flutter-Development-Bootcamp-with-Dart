# The Complete Flutter Development Bootcamp with Dart

This repository contains my progress and projects from "The Complete Flutter Development Bootcamp with Dart" course by Dr. Angela Yu.

## About the Course

This comprehensive course covers all the essential aspects of Flutter development, from the very basics to advanced topics. You'll learn how to build beautiful, performant, and cross-platform mobile applications using Flutter and Dart.

## Getting Started

1.  **Clone the repository:**

    ```bash
    git clone [repository URL]
    ```

2.  **Navigate to the project folder:**

    ```bash
    cd [project folder]
    ```

3.  **Install dependencies:**

    ```bash
    flutter pub get
    ```

4.  **Run the app:**

    ```bash
    flutter run
    ```

## Prerequisites

* Flutter SDK installed and configured.

* Android Studio or Xcode (depending on your target platform).
